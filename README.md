## 2. Início Rápido

### 2.1. Instalando Dependências

Clone o projeto em sua máquina e instale as dependências com o comando:

> - Pelo yarn:
>
> ```shell
> yarn
> ```
>
> - Pelo npm:
>
> ```shell
> npm install
> ```

### 2.2. Rodar Projeto

Rode o projeto com o comando:

> - Pelo yarn:
>
> ```shell
> yarn run dev
> ```
>
> - Pelo npm:
>
> ```shell
> npm run dev
> ```
